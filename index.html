Comandos do Git

git init

Inicializa um novo repositório Git no diretório atual.

git clone [url]

Clona um repositório remoto para o seu computador.

git add [arquivo]

Adiciona arquivos ao índice (staging area) para serem commitados.

git commit -m "mensagem"

Salva as alterações no repositório local com uma mensagem descritiva.

git push origin [branch]

Envia as alterações locais para o repositório remoto na branch especificada.

git pull

Atualiza o repositório local com as alterações do repositório remoto.

git branch

Lista todas as branches do repositório. Use git branch [nome] para criar uma nova branch.

git checkout [branch]

Muda para a branch especificada. Use git checkout -b [nome] para criar e mudar para uma nova branch.

git merge [branch]

Combina a branch especificada com a branch atual.

git status

Mostra o estado atual do repositório, incluindo arquivos modificados e no staging.

GitHub e Integração

git remote add origin [url]

Conecta o repositório local a um repositório remoto no GitHub.

git push -u origin [branch]

Envia a branch local para o GitHub e configura o rastreamento.

GitHub Pages

Para hospedar um site, crie um repositório chamado username.github.io, adicione um index.html e configure o GitHub Pages em Settings > Pages.

Pull Requests

No GitHub, crie um Pull Request (PR) para propor alterações de uma branch para outra. Acesse a aba "Pull Requests" no repositório e clique em "New pull request".

Fork e Clone

Fork um repositório no GitHub para criar uma cópia no seu perfil. Use git clone para baixá-lo e contribuir.
